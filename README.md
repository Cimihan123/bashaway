## What is this 

This is a script that automates daily basis tasks. You can do day-today tasks with this script.

## How To intall

```
cd ~
apt update
git clone https://github.com/sliit-foss/bashaway.git
cd bashaway
chmod +x install.sh
./install.sh
```
### How to run it

```
bshw
```
If you remainder task number

```
bshw < TaskNumber >
```

### How to uninstall

``` 
cd ~/.local/share/bashaway 
./uninstall.sh
```
